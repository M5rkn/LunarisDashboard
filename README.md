Dashboard Analytics

Панель управления для визуализации данных с графиками и экспортом отчётов.

Tech Stack

Vue.js 3 - фреймворк
Chart.js - графики и диаграммы
Firebase - бэкенд и база данных
Vuetify - UI компоненты
Vite - сборщик
Pinia - управление состоянием



Функционал

Дашборд - карточки метрик, графики, таблица заказов
Графики - линейные и круговые диаграммы
Отчёты - генерация и экспорт отчётов
Настройки - конфигурация приложения и Firebase
Экспорт - выгрузка данных в CSV
